## Usage

```jsx
import { Drawer, Button } from 'tamagui'

export default () => (
  <Drawer contents="Text in drawer">
    <Button>Hover to open</Button>
  </Drawer>
)
```

### Drawer props

Drawers extend Stack views inheriting all the [Tamagui standard props](/docs/intro/props), plus:
