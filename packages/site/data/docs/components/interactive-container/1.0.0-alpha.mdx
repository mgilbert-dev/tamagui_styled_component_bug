## Usage

```jsx
import { InteractiveContainer } from 'tamagui'

export default () => (
  <InteractiveContainer>
    Lorem ipsum
  </InteractiveContainer>
)
```

### Sizing

Sizing interactiveContainers provides a unique challenge especially for a compiler, because you need to adjust many different properties - not just on the outer frame, but on the text wrapped inside. Tamagui supports adjusting the padding, border radius, and font size all in one with the `size` prop.

```jsx
import { InteractiveContainer } from 'tamagui'

export default () => (
  <InteractiveContainer size="$6">
    Lorem ipsum
  </InteractiveContainer>
)
```

Given your theme defines a size `6`, the interactiveContainer will adjust all of the properties appropriately. You can also pass a plain number to get an arbitrary size.
