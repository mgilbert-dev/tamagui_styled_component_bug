## Usage

Text is the basic text view in Tamagui, but it doesn't automatically apply any styling. If you want to have your text views inherit colors, fonts and

```jsx
import { Button } from 'tamagui'

export default () => (
  <Button>
    Lorem ipsum
  </Button>
)
```
