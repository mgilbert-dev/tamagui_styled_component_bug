# Feather Icons

<Notice>
  Deprecated in favor of [lucide-icons](/docs/components/lucide-icons), a strict superset of feather
  icons that is also tree-shakeable.
</Notice>

### Usage

Add `@tamagui/lucide-icons` to your package.json and import them directly for use as React elements.
